                                    MOVEMENT CONTROL LOGIC
                                    ======================

                    ┌─────────────────────────────────────────────────────────┐
                    │                FORWARD MOVEMENT                         │
                    │                                                         │
                    │  Motor1: OUT1(+) OUT2(-)  →  Clockwise                │
                    │  Motor2: OUT3(+) OUT4(-)  →  Clockwise                │
                    │                                                         │
                    │  GPIO27=LOW, GPIO26=HIGH  (Motor1 Forward)             │
                    │  GPIO33=LOW, GPIO25=HIGH  (Motor2 Forward)             │
                    └─────────────────────────────────────────────────────────┘

                    ┌─────────────────────────────────────────────────────────┐
                    │                REVERSE MOVEMENT                         │
                    │                                                         │
                    │  Motor1: OUT1(-) OUT2(+)  →  Counter-Clockwise        │
                    │  Motor2: OUT3(-) OUT4(+)  →  Counter-Clockwise        │
                    │                                                         │
                    │  GPIO27=HIGH, GPIO26=LOW  (Motor1 Reverse)             │
                    │  GPIO33=HIGH, GPIO25=LOW  (Motor2 Reverse)             │
                    └─────────────────────────────────────────────────────────┘

                    ┌─────────────────────────────────────────────────────────┐
                    │                 LEFT TURN                               │
                    │                                                         │
                    │  Motor1: STOP                                          │
                    │  Motor2: OUT3(+) OUT4(-)  →  Clockwise                │
                    │                                                         │
                    │  GPIO27=LOW, GPIO26=LOW   (Motor1 Stop)               │
                    │  GPIO33=LOW, GPIO25=HIGH  (Motor2 Forward)             │
                    └─────────────────────────────────────────────────────────┘

                    ┌─────────────────────────────────────────────────────────┐
                    │                RIGHT TURN                               │
                    │                                                         │
                    │  Motor1: OUT1(+) OUT2(-)  →  Clockwise                │
                    │  Motor2: STOP                                          │
                    │                                                         │
                    │  GPIO27=LOW, GPIO26=HIGH  (Motor1 Forward)             │
                    │  GPIO33=LOW, GPIO25=LOW   (Motor2 Stop)                │
                    └─────────────────────────────────────────────────────────┘
